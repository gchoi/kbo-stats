# kbo-stats
Record of Hitters for Last 10 Games in KBO (Korea Baseball League) by web-crawling [KBO Website](https://www.koreabaseball.com)

## Prerequisites
* Python Python 3.6
* Beautifulsoup4

## Usage
```
$ python kbo-stats.py
```

## Results
HTML file will be created in `results`(index.html) path.
